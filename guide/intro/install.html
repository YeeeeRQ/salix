# 安装使用

# 添加至项目

打开终端运行下列命令:

npm install salix-ui

或

yarn add salix-ui

# 开始使用

请先安装本组件库

然后在您的代码中写入下面的代码

import {Button, Switch, Dialog, Tabs, Tab} from "salix-ui";
